SONGS FOLDER INSTRUCTIONS
==========================

Place your songs/audio files here with the following names:
- landing.mp3 (for the main calendar page)
- day1.mp3 (for February 1)
- day2.mp3 (for February 2)
- day3.mp3 (for February 3)
- day4.mp3 (for February 4)
- day5.mp3 (for February 5)
- day6.mp3 (for February 6)
- day7.mp3 (for February 7)
- day8.mp3 (for February 8)
- day9.mp3 (for February 9)
- day10.mp3 (for February 10)
- day11.mp3 (for February 11)
- day12.mp3 (for February 12)
- day13.mp3 (for February 13)
- day14.mp3 (for February 14 - Valentine's Day)

Supported formats: .mp3, .wav, .ogg
Each song will play automatically (with fallback to user interaction) when the day page is opened.
Volume is set to 30% by default and loops continuously.
If songs are not added, no background music will play.
